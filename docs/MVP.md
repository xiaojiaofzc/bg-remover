# Image Background Remover - MVP 需求文档

## 1. 产品定位

### 1.1 产品定义

| 项目 | 内容 |
|------|------|
| **产品名称** | 图片背景去除器 |
| **产品类型** | 在线工具 / SaaS |
| **产品定位** | 快速去除图片背景的在线工具，让用户无需PS即可获得透明背景图片 |

### 1.2 目标用户

| 用户群体 | 痛点 | 我们的价值 |
|----------|------|------------|
| **电商卖家** | 淘宝/拼多多商品图需要白底或透明背景 | 5秒出图，上传即用 |
| **设计师** | 抠图耗时，需要快速获取素材 | AI自动抠，专注创意 |
| **自媒体创作者** | 封面、配图需要透明背景 | 简单快捷，无需学习PS |
| **普通用户** | 证件照换底、头像处理 | 免费额度够用，操作简单 |

### 1.3 核心价值

- ⚡ **快**：5秒完成，无需安装软件
- 💰 **省**：免费额度足够个人使用
- 🎯 **准**：AI自动识别，精准抠图
- 📱 **简**：拖拽上传，点点点就完成

### 3.4 前端组件

#### 上传组件

```javascript
// 上传区域 HTML
<div class="upload-area" id="uploadArea">
  <input type="file" id="fileInput" accept="image/jpeg,image/png">
</div>

// 拖拽处理
uploadArea.addEventListener('drop', (e) => {
  e.preventDefault();
  const file = e.dataTransfer.files[0];
  handleFile(file);
});

// 文件验证
function validateFile(file) {
  const validTypes = ['image/jpeg', 'image/png'];
  const maxSize = 10 * 1024 * 1024; // 10MB
  
  if (!validTypes.includes(file.type)) {
    throw new Error('仅支持 JPG/PNG 格式');
  }
  if (file.size > maxSize) {
    throw new Error('文件大小不能超过 10MB');
  }
}
```

#### API 调用

```javascript
async function removeBackground(file) {
  const formData = new FormData();
  formData.append('image', file);
  
  const response = await fetch('/remove-bg', {
    method: 'POST',
    body: formData
  });
  
  if (!response.ok) {
    const error = await response.json();
    throw new Error(error.message);
  }
  
  // 返回 Blob
  return await response.blob();
}
```

## 9. 技术风险

### 9.1 风险清单

| 序号 | 风险 | 等级 | 影响 |
|------|------|------|------|
| R1 | Remove.bg API 不稳定 | 高 | 服务不可用 |
| R2 | 免费额度用完 | 中 | 用户流失 |
| R3 | 并发过高被限流 | 中 | 部分请求失败 |
| R4 | 图片太大处理超时 | 低 | 用户体验差 |

### 9.2 应对方案

#### R1: Remove.bg API 不稳定

```
风险描述: API 响应慢或返回错误
影响: 用户无法完成背景去除

应对措施:
├── 1. 添加超时机制 (5秒)
├── 2. 重试机制 (最多3次)
├── 3. 备用 API (ClipDrop / 阿里云)
└── 4. 友好的错误提示

代码示例:
async function callAPIWithRetry(imageData, maxRetries = 3) {
  for (let i = 0; i < maxRetries; i++) {
    try {
      return await removeBackground(imageData);
    } catch (err) {
      if (i === maxRetries - 1) throw err;
      await sleep(1000 * (i + 1)); // 指数退避
    }
  }
}
```

#### R2: 免费额度用完

```
风险描述: Remove.bg 免费版每月 50 张
影响: 免费用户无法使用

应对措施:
├── 1. 实时监控使用量
├── 2. 接近额度时提示用户
├── 3. 接入备用 API (阿里云/腾讯云)
├── 4. 引导 Pro 版转化
└── 5. 考虑自建模型 (长期)

备选 API:
├── 阿里云 - 图像分割: 每月 500 次免费
├── 腾讯云 - 人像分割: 每月 1000 次免费
└── ClipDrop API: 100 次免费/天
```

#### R3: 并发过高被限流

```
风险描述: Cloudflare Workers 或 API 并发限制
影响: 高峰期请求失败

应对措施:
├── 1. 请求排队机制
├── 2. 限流提示
├── 3. 升级付费套餐
└── 4. 缓存优化

Cloudflare 限制:
├── 1000 请求/分钟 (免费版)
├── 队列处理避免并发冲击
└── 适当返回 429 提示
```

#### R4: 图片太大处理超时

```
风险描述: 大图片处理时间过长
影响: 用户等待时间长，可能超时

应对措施:
├── 1. 前端限制图片大小 (10MB)
├── 2. 前端压缩图片
├── 3. 服务端超时设置 (30秒)
└── 4. 友好的超时提示

前端压缩示例:
function compressImage(file, maxSize = 1920) {
  return new Promise((resolve) => {
    const canvas = document.createElement('canvas');
    const ctx = canvas.getContext('2d');
    const img = new Image();
    
    img.onload = () => {
      const scale = Math.min(maxSize / img.width, maxSize / img.height);
      canvas.width = img.width * scale;
      canvas.height = img.height * scale;
      ctx.drawImage(img, 0, 0, canvas.width, canvas.height);
      canvas.toBlob(resolve, 'image/jpeg', 0.8);
    };
    img.src = URL.createObjectURL(file);
  });
}
```
